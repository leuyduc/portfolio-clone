# Portfolio Clone

DAS-inspired portfolio/studio landing page built for layout and motion research. The project recreates the feel of an editorial design-school website without copying original branding, logo, proprietary images, or copy.

## Stack

- Vite
- HTML/CSS
- GSAP + ScrollTrigger
- Lenis smooth scroll

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Project Structure

```text
index.html
src/main.js
src/styles.css
src/animations/
```

## Notes

This is a study clone. Replace placeholder content/assets before using it for a real brand or commercial site.
